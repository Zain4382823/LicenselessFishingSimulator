# LicenselessFishingSimulator

# CONTROL SCHEME:

## WASD -> Movement
## Arrow Keys -> Navigate in menu
## F -> Enter / Exit Fishing Mode
## C -> Catch a fish
## E -> Open / Close Inventory
## Enter -> Interact
